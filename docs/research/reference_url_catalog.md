## Quick Lookup — Which EPSG for Cape Town?

| Code | Name | Use When |
|---|---|---|
| **EPSG:4326** | WGS84 Geographic | Store in PostGIS, serve as GeoJSON, all web APIs |
| **EPSG:3857** | Web Mercator | CartoDB/OSM basemap tile rendering internally |
| **EPSG:22291** | Lo19 (Hartebeesthoek94) | Raw City of Cape Town source data — transform to 4326 on import via `ST_Transform(geom, 4326)` |
