# User Management & Admin Panel Guide

Welcome to the Tenant Admin Panel. This guide explains how to manage your team and monitor activity.

## Cyber-Crayon Interface

The Admin Panel follows the "Cyber-Crayon" design theme:
-   **Dark Mode**: Deep gray backgrounds with high-contrast text.
-   **Neon Accents**: Active tabs and status indicators use vibrant neon colors (e.g., Electric Blue, Neon Magenta).
-   **Crayon Styling**: Soft borders and "sketchy" accents to differentiate interactive elements.

## Panel Navigation

The Admin Panel is divided into three primary sections:

### 1. Users Tab
Manage active users within your tenant.
-   **View All Users**: See a list of all registered team members.
-   **Role Management**: Change user roles (Viewer, Analyst, Power User, Admin) using the dropdown menus.
-   **Account Status**: Deactivate or re-enable accounts.

### 2. Invites Tab
Invite new members to join the platform.
-   **Send Invite**: Enter the email address and select an initial role.
-   **Pending Invites**: Track who has not yet accepted their invitation.
-   **Resend/Revoke**: Manage existing invites.

### 3. Audit Log Tab
Monitor all system activity for security and transparency. The log tracks the following events:
-   `login`: When a user signs into the system.
-   `role_change`: When a user's permission level is modified.
-   `user_invite`: When a new invitation is sent.

*The Audit Log is read-only and provides a timestamped history of actions taken by users and admins.*
